# LangChain RAG
## Purpose  
Chatbot application to have Q&A for regulatory documents
## Implementation  
LangChain RAG with a vector database, as Hybrid LLM with the following components:  
1. Python Version 3.12  
2. langchain>=1.1.0  
3. langchain-core  
4. langchain-community  
5. langchain-openai  
6. faiss-cpu  
7. langchain-text-splitters  
8. python-dotenv  
9. pillow
